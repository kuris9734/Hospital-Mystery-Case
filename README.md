# 城郊医院谜案 - 互动解密游戏

## 游戏简介

《城郊医院谜案》是一款日式漫画风格的网页解密类互动游戏。玩家扮演一名调查员，深入调查一家城郊老旧医院中发生的神秘事件，通过收集线索、破解密码、与角色对话来逐步揭开真相。

## 游戏特色

- **日式漫画风格**：精美的角色立绘和场景设计
- **悬疑推理剧情**：多线索、多结局的复杂故事
- **互动解密**：密码破解、线索收集、场景探索
- **角色互动**：9位独特角色，丰富的对话系统
- **多结局设定**：根据玩家的选择和解谜进度触发不同结局

## 核心玩法

### 1. 线索收集
- 在7个不同场景中搜寻线索
- 线索会自动存入背包系统
- 关键线索可以解锁新场景

### 2. 密码解谜
- 多处密码锁需要破解
- 密码线索隐藏在对话和场景中
- 成功破解可获得重要证据

### 3. 角色对话
- 与9位角色进行深度对话
- 不同选择影响剧情走向
- 关键信息往往藏在对话细节中

### 4. 场景探索
- 7个精心设计的游戏场景
- 每个场景都有独特的互动元素
- 场景间存在逻辑关联

## 角色介绍

### 主要角色
- **陈橘子**：医院院长，表面冷静，实则隐藏着可怕秘密
- **婷护士**：失踪的关键人物，留下了重要线索
- **安保人员A**：夜班保安，知道很多深夜的异常情况
- **患者X**：长期住院患者，似乎知道地下室的秘密

### 其他角色
- 实习医生、档案管理员、维修工、保洁阿姨、医院设计师

## 游戏场景

1. **医院走廊** - 游戏起始点，发现基础线索
2. **手术室** - 隐藏着被修改的手术记录
3. **住院部三层** - 部分病房被神秘封锁
4. **院长室** - 陈院长的私人办公室，藏有重要文件
5. **监控室** - 可以查看异常时间的监控记录
6. **地下电梯通道** - 需要特殊条件才能进入
7. **实验档案室** - 最终的真相揭露地点

## 结局设定

### 真相揭露结局（最佳结局）
- 收集足够证据揭露医院的非法人体实验
- 救出婷护士和其他失踪患者
- 陈院长被绳之以法

### 逃离结局（普通结局）
- 带着部分线索逃离医院
- 但证据不足，无法阻止实验继续

### 失踪结局（坏结局）
- 在调查过程中神秘失踪
- 成为又一个受害者

## 游戏操作

### 基本操作
- **鼠标点击**：与场景和角色互动
- **背包按钮**：查看已收集的线索
- **地图按钮**：在不同场景间切换
- **保存按钮**：保存游戏进度

### 界面说明
- **左侧主区域**：显示当前场景和对话
- **右侧背包**：显示收集的线索和进度
- **底部对话框**：显示对话内容和选择选项

## 游戏技巧

1. **仔细探索**：每个场景都有多个可互动物品
2. **记录信息**：重要的密码和线索需要记住
3. **多对话**：与角色多次对话可能获得新信息
4. **逻辑推理**：结合多个线索来解开谜题
5. **保存进度**：定期保存，避免丢失进度
